## Button

Button is a graphical control element that provides the user a simple way to trigger an event.

export const Template = args => <Button {...args} />;

### Type

By changing the `btnType` prop you can use different styles of the button.

### Variant

Choose variant of the Button by changing the `variant` prop.<br />
**Text** buttons are typically used for _less-pronounced_ actions.<br />
**Outlined** buttons are _medium-emphasis_.<br />
**Contained** buttons are _high-emphasis_.

<Canvas>
	<Story name="TextVariant" args={{ title: 'Text', variant: 'text' }} argTypes={TextbuttonArgTypes}>

	</Story>

### Button Props

| Name          | Required |                                   Type                                   | DefaultValue | Description                                                                                                                                                              |
| :------------ | :------: | :----------------------------------------------------------------------: | :----------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `title`       |    -     |                                **string**                                |              | Specify the text content of the Button                                                                                                                                   |
| `type`        |    -     |              **Union**<br/>'button' \| 'submit' \| 'reset'               |              | Optional prop to specify the type of the Button                                                                                                                          |
| `size`        |    -     |               **Union**<br/>'small' \| 'medium' \| 'large'               |              | Choosing a size of the Button                                                                                                                                            |
| `btnType`     |    -     | **Union**<br/>'primary' \| 'secondary' \| 'error' \| 'success' \| 'info' |              | Choosing a semantic type of the Button                                                                                                                                   |
| `variant`     |    -     |            **Union**<br/>'text' \| 'outlined' \| 'contained'             | 'contained'  | Choosing a variant of the Button. Text buttons are typically used for less-pronounced actions, outlined buttons are medium-emphasis, contained buttons are high-emphasis |
| `disabled`    |    -     |                               **boolean**                                |    false     | Specify whether the Button should be disabled, or not                                                                                                                    |
| `startIcon`   |    -     |                             **JSX.Element**                              |              | Adding an icon to the Button _before text content_                                                                                                                       |
| `endIcon`     |    -     |                             **JSX.Element**                              |              | Adding an icon to the Button _after text content_                                                                                                                        |
| `icon`        |    -     |                             **JSX.Element**                              |              | Setting an icon to the Button                                                                                                                                            |
| `clickAction` |    -     |                               ( ) => void                                |              | Allows you to call a function and perform an action when an element is clicked                                                                                           |
