### Generic Modal

Create custom modal by adding content via `children` property, set styles for modal content by `contentStyles` property. Also you can set your own styles for modal title and specify whether the button section and tertiary button should be displayed, or not. Use `titleStyles`, `showButtonsSection` and `showTertiaryBtn` props for this task.

<Canvas>
	<Story
		name="GenericModal"
		args={{
			title: 'Create your accaunt',
			showButtonsSection: true,
			titleStyles: {
				color: '#0F0F0F',
				fontSize: '24px',
				margin: '0',
				padding: '0',
			},
			contentStyles: {
				display: 'grid',
				gridTemplateColumns: 'repeat(2, 1fr)',
				gridTemplateRows: 'repeat(3, 1fr)',
				gap: '15px',
				padding: '20px 0',
				borderTop: '1px solid #D9D9D9',
				borderBottom: '1px solid #D9D9D9',
				margin: '10px 0 15px'
			},
		}}
	>
		{GenericModalTemplate.bind({})}
	</Story>
</Canvas>

### Modal Props

| Name                 | Required |       Type        | DefaultValue | Description                                                                               |
| :------------------- | :------: | :---------------: | :----------: | :---------------------------------------------------------------------------------------- |
| `title`              |    +     |    **string**     |              | Specify the title of the modal component.                                                 |
| `children`           |    +     |  **JSX.Element**  |              | Property which contains child elements defined within the generic modal component.        |
| `primaryBtnTxt`      |    +     |    **string**     |    'Save'    | Specify the text content of the primary button.                                           |
| `primaryBtnClick`    |    +     |    ( ) => void    |              | Allows you to call a function and perform an action when the primary button is clicked.   |
| `secondaryBtnTxt`    |    +     |    **string**     |   'Reset'    | Specify the text content of the secondary button.                                         |
| `secondaryBtnClick`  |    +     |    ( ) => void    |              | Allows you to call a function and perform an action when the secondary button is clicked. |
| `showTertiaryBtn`    |    -     |    **boolean**    |     true     | Specify whether the tertiary button should be displayed, or not.                          |
| `tertiaryBtnTxt`     |    -     |    **string**     |   'Cancel'   | Specify the text content of the tertiary button.                                          |
| `tertiaryBtnClick`   |    -     |    ( ) => void    |              | Allows you to call a function and perform an action when the tertiary button is clicked.  |
| `titleStyles`        |    -     | **CSSProperties** |              | Set styles for modal title.                                                               |
| `contentStyles`      |    -     | **CSSProperties** |              | Set styles for modal content.                                                             |
| `showButtonsSection` |    -     |    **boolean**    |     true     | Specify whether the button section should be displayed, or not.                           |
