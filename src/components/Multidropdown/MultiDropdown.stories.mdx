#### Using items prop

By passing `items` prop you can create multiple dropdown

<Canvas>
	<Story name="Using items prop" args={{ items: ExampleItems }}>
		{Template.bind({})}
	</Story>
</Canvas>

#### Using children prop

By passing `children` prop you can create dropdown

<Canvas>
	<Story name="Using children prop" args={{children: <div style={{padding: '30px'}}>{textExample}</div>}}>
		{Template.bind({})}
	</Story>
</Canvas>

#### Position reversed

By passing `position` prop you can change expand direction of dropdown

<Canvas>
	<Story name="Reversed" args={{position: 'reversed'}} parameters={{
		layout: 'centered'
	}}>
		{Template.bind({})}
	</Story>
</Canvas>

#### Expanded

By passing `isExpanded` prop you can create dropdown that will be expanded by default

<Canvas>
	<Story name="Expanded" args={{isExpanded: true}}>
		{Template.bind({})}
	</Story>
</Canvas>


#### Styles

By passing `dropdownStylesRoot` and prop you can change default style of dropdown or dropdown content

<Canvas>
	<Story name="StyledRoot" args={{dropdownStylesRoot: {backgroundColor: 'green', color: '#fff', '& svg': {
				color: '#fff',
			}},
	}}>
		{Template.bind({})}
	</Story>
</Canvas>
